# TimeVault Chrome Extension Icons

This directory should contain the following icon files:
- icon-16.png  (16x16 pixels)
- icon-32.png  (32x32 pixels)  
- icon-48.png  (48x48 pixels)
- icon-128.png (128x128 pixels)

## Creating Icons

You can create icons using any image editing software. The icon should:
- Use the TimeVault brain/vault theme
- Be clear and recognizable at small sizes
- Use the purple gradient color scheme (#667eea to #764ba2)

## Temporary Solution

For development, you can:
1. Create simple colored squares with the TimeVault logo
2. Use online icon generators
3. Export from design software like Figma or Canva

## Suggested Design

- Background: Purple gradient
- Icon: Brain or vault symbol in white
- Text: "TV" monogram for smaller sizes
